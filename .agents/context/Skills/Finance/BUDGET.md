# Rule — Budget & expenses

1. Infrastructure costs (server, DB, CDN) tracked monthly, tagged per app on the cloud provider
2. Alert threshold: monthly spend exceeds 110% of the 3-month average → report immediately in #finance
3. New services/SaaS >$50/month: require founder approval before signing up
4. Trials/PoCs must have a cancellation reminder scheduled at sign-up to avoid forgotten charges
5. AI API costs (LLM, embedding) kept as a separate line item, reviewed weekly
6. All invoices saved to Drive `Finance/Invoices/<year>/<month>` within 3 days
