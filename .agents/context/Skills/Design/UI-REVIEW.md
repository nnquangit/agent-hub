# Skill: UI Review

Checklist when reviewing MyApp UI:

- [ ] Follows the design system ([DESIGN-SYSTEM.md](../../Rules/Designer/DESIGN-SYSTEM.md))
- [ ] Matches UI guidelines ([UI-GUIDELINES.md](../../Rules/Designer/UI-GUIDELINES.md))
- [ ] Responsive: mobile / tablet / desktop
- [ ] Dark mode colors don't break
- [ ] Loading / empty / error states all covered
- [ ] Accessibility: focus ring, contrast, labels for inputs
